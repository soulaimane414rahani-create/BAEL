# Application de Calculs BAEL

Application web pour les calculs de béton armé selon les normes BAEL.

## Fonctionnalités
- Calcul des sections de béton armé
- Vérification ELS et ELU
- Interface web intuitive

## Installation
1. Installer les dépendances : `pip install -r requirements.txt`
2. Lancer l'application : `python app.py`
3. Ouvrir : http://localhost:5000
